# Nimble Chat

A one-to-one real-time messaging system featuring secure JWT authentication and persistent message history.

---

## Features

- **Real-time Communication**: Instant message delivery.
- **Secure Access**: JWT-based authentication.
- **Persistence**: All messages are saved and retrievable.

---

## Tech Stack

- **React**
- **TypeScript**
- **Express.js**
- **Socket.io**
- **MongoDB**

---

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- yarn

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/harshk341/nimble-chat.git
   ```

2. Navigate to the project directory:

   ```bash
   cd nimble-chat
   ```

3. Install Root Dependencies (used for running both client and server concurrently):

   ```bash
   yarn install
   ```

4. Install Project Dependencies:

   ```bash
   yarn install:all
   ```

5. Start the development (server and client):

   ```bash
   yarn dev
   ```

6. Open your browser and visit `http://localhost:5173`
